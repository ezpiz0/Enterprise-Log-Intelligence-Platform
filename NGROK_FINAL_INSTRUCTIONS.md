## ⚡ ЧТО ДЕЛАТЬ СЕЙЧАС (3 простых шага)

### Шаг 1: Установите ngrok (2 минуты)

```
1. Откройте: https://ngrok.com/download
2. Скачайте: Windows 64-bit
3. Распакуйте ngrok.exe в C:\ngrok\
```

### Шаг 2: Настройте Authtoken (30 секунд)

Откройте **PowerShell** и выполните:

```powershell
C:\ngrok\ngrok.exe config add-authtoken 33xJiXbggbf0VkJQYYS0Q1Rt7IF_4N9bEXBDXbBDvzMjvEPLy
```

**Результат:** ✅ `Authtoken saved to configuration file`

### Шаг 3: Запустите Всё (2 минуты)

#### Вариант A: Автоматический (Проще!)

1. **Двойной клик** на `start-server.bat`
2. **Двойной клик** на `start-ngrok.bat`
3. **Скопируйте URL** из окна ngrok

#### Вариант B: Вручную

**Окно 1 - PowerShell:**
```powershell
cd D:\Downloads\FASTAPIGITPROJECT
python main.py
```

**Окно 2 - PowerShell:**
```powershell
C:\ngrok\ngrok.exe http 8001
```

## 🐛 Часто Задаваемые Вопросы

### Q: "ngrok not found" при запуске

**A:** Используйте полный путь:
```powershell
C:\ngrok\ngrok.exe http 8001
```

---

### Q: "Connection refused" в ngrok

**A:** Сначала запустите сервер!
```powershell
python main.py
```
Дождитесь: `Uvicorn running on http://127.0.0.1:8001`

---

### Q: Друзья видят предупреждение ngrok

**A:** Это нормально для бесплатного плана!
Скажите им нажать **"Visit Site"** для продолжения.

---

### Q: URL не работает / изменился

**A:** URL меняется при каждом перезапуске ngrok!
Скопируйте новый URL из окна ngrok и отправьте друзьям.

---

### Q: Можно ли получить постоянный URL?

**A:** Да, но нужен платный план ngrok ($8/месяц).
Для демонстрации бесплатного хватает!

---

### Q: Сколько человек могут подключиться?

**A:** Бесплатный план:
- 1 туннель одновременно
- 40 подключений в минуту
- Для демонстрации команде из 5-10 человек хватит
