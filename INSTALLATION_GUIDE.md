# Руководство по установке и запуску проекта FastAPI Processor

## Описание проекта
Это приложение для анализа логов ИТ-инфраструктуры с использованием машинного обучения. Приложение принимает ZIP-архивы с логами и CSV файлами, анализирует их и предоставляет интерактивный дашборд с результатами.

## Системные требования
- Python 3.8 или выше
- Windows/Linux/macOS
- Минимум 4 ГБ RAM
- Свободное место на диске: 2 ГБ

## Установка зависимостей

### 1. Установите Python
Убедитесь, что у вас установлен Python 3.8+:
```bash
python --version
```

### 2. Установите зависимости
Откройте терминал в папке с проектом и выполните:

```bash
pip install fastapi uvicorn pandas sentence-transformers scikit-learn openpyxl
```

Или создайте файл `requirements.txt` и установите все зависимости одной командой:

```bash
pip install -r requirements.txt
```

## Запуск приложения

### 1. Откройте терминал в папке проекта

### 2. Запустите сервер
```bash
python -m uvicorn main:app --host 127.0.0.1 --port 8002
```

### 3. Откройте браузер
Перейдите по адресу: http://127.0.0.1:8002

## Использование

### 1. Выберите модель
- **Быстрая модель** - для быстрого анализа (рекомендуется)
- **Точная модель** - для максимальной точности (медленнее)

### 2. Загрузите файлы
- Перетащите ZIP-архив в зону загрузки
- Архив должен содержать:
  - `.txt` файлы с логами
  - `anomalies_problems` - база знаний (CSV, XLSX или XLS)

### 3. Запустите анализ
- Нажмите кнопку "Запустить Анализ"
- Дождитесь завершения обработки

### 4. Просмотрите результаты
- Автоматически откроется дашборд с результатами
- Скачайте ZIP-архив с отчетами

## Возможные проблемы

### Порт занят
Если порт 8002 занят, используйте другой:
```bash
python -m uvicorn main:app --host 127.0.0.1 --port 8003
```

### Ошибки зависимостей
Убедитесь, что все пакеты установлены:
```bash
pip install --upgrade pip
pip install -r requirements.txt
```

### Проблемы с кэшем браузера
- Нажмите Ctrl + F5 для принудительного обновления
- Очистите кэш браузера
- Откройте в режиме инкогнито

## Поддержка
При возникновении проблем проверьте:
1. Версию Python (должна быть 3.8+)
2. Установлены ли все зависимости
3. Свободен ли порт 8002
4. Корректность структуры ZIP-архива
