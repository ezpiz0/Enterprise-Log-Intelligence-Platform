🚀 БЫСТРЫЙ ЗАПУСК:
━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━

1. УСТАНОВКА ЗАВИСИМОСТЕЙ:
   
   pip install -r requirements.txt


2. ЗАПУСК СЕРВЕРА:
   
   python main.py
   
   Сервер запустится на: http://127.0.0.1:8001


3. ИСПОЛЬЗОВАНИЕ:
   
   ✓ Откройте браузер: http://127.0.0.1:8001
   ✓ Выберите модель (Быстрая или Точная)
   ✓ Загрузите ZIP-архив с логами (*.txt файлы + anomalies_problems.csv)
   ✓ Нажмите "Запустить Анализ"
   ✓ Автоматически откроется дашборд с результатами

🚀 УСКОРЕНИЕ С GPU (ОПЦИОНАЛЬНО):
━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━

✨ АВТОМАТИЧЕСКОЕ ОПРЕДЕЛЕНИЕ GPU!
   Система автоматически определяет наличие NVIDIA GPU и использует его.
   Если GPU недоступен - автоматически используется CPU.

📥 ДЛЯ ИСПОЛЬЗОВАНИЯ GPU:
   1. Установите драйверы NVIDIA и CUDA Toolkit
   2. Установите PyTorch с CUDA:
      pip install torch torchvision torchaudio --index-url https://download.pytorch.org/whl/cu118
   3. Запустите check_gpu.py для проверки:
      python check_gpu.py

📈 УСКОРЕНИЕ:
   • Быстрая модель: CPU ~30 сек → GPU ~5 сек
   • Точная модель: CPU ~120 сек → GPU ~20 сек
   • Общее ускорение: 5-10x

⚡ При запуске вы увидите:
   🚀 Используется GPU: NVIDIA GeForce RTX 3080 (если GPU есть)
   ⚠️  GPU не обнаружен, используется CPU (если GPU нет)
